# Internship-14-Millionaire
**React App which emulates "Who Wants to Be a Millionaire"**

---

## Features
- Continue the game (if you refresh the tab or close the browser completely)
- Giving up and taking the last prize allowed
- Joker support (50/50, skip question, swap question)
- Score tracking and progress display
---

## Requirements
- Node.js (v18+ recommended)
- npm (Node Package Manager)

---

## Installation
1. Clone the repository:
```
git clone git@github.com:DorianLeci/Internship-14-Millionaire.git
```

2.Navigate into the project folder:
```
cd Internship-14-Millionaire/Millionaire
```

3.Install dependencies:
```
npm install
```

4.Run app:
```
npm run dev
```
